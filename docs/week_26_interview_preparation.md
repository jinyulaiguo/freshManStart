*   **核心知识点**：

## Day 179：STAR 面试故事（四）：Harness 三层安全防御与防失控沙箱机制
*   **核心知识点**：

## Day 180：高级 System Design：设计支持日活百万的多租户高并发 Agent 云平台
*   **核心知识点**：
    *   **系统设计要素**：
        - 架构分层：网关层（路由/限流/鉴权） $	o$ 任务分发层（消息队列削峰） $	o$ Agent 执行引擎集群（无状态部署/Redis checkpoint 同步） $	o$ 存储层（向量数据库/持久化关系库）。
