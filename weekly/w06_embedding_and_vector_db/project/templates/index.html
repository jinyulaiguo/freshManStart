DocumentParser (异构格式解析)

                            
自动路由文件格式并进行结构化解析，保留 Markdown 章节及 Python 代码块结构。

                            

                        

                        
                        

                            
                            
TextCleaner (格式去噪与规范化)

                            
剔除控制字符及 HTML 冗余编码，提取纯净文本，并输出噪声清洗率。

                            

                        

                        
                        

                            
                            
ChunkEngine (语义滑动窗口分块)

                            
执行基于 Token 颗粒度的智能滑动切片，并对内容计算唯一 SHA-256 哈希指纹。

                            

                        

                        
                        

                            
                            
EmbeddingPipeline (限流 API 向量化)

                            
进行多协程并发拉取向量。包含客户端信号量控制，防 429 报错自适应退避自愈。

                            

                        

                        
                        

                            
                            
Qdrant VectorStore (多集合写入与索引构建)

                            
将向量批量 upsert 入库，并在底层创建 Payload 元数据索引及 HNSW 检索树。
